# **App Name**: Myanmar 2D Lottery Tracker

## Core Features:

- Historical Data Storage: Automatically search for and update the historical 2D numbers daily and save them in the Firestore database.
- Data Display: Display the current (most recent) 2D number.
- Number Pattern Analysis: Use AI to identify patterns in the lottery data for the last 4 weeks, providing potential insights into frequently drawn numbers. Act as a tool.

## Style Guidelines:

- Primary color: Deep purple (#673AB7), reflecting the subtle mystery and anticipation associated with lotteries.
- Background color: Very light purple (#F3E5F5), creates a calm backdrop.
- Accent color: Analogous color Pink (#E91E63) for interactive elements to highlight insights.
- Body and headline font: 'PT Sans', sans-serif with a modern and little warmth, ensuring easy readability for all users.
- Simple, clean icons for navigation, such as a calendar icon for date selection.
- Clean and intuitive layout, prioritizing clear display of the most recent results and number pattern analysis.
